Portfolio Design System

About My System

Welcome to the design system for my portfolio. In this system I include guidelines for typography, color, branding elements, page layouts, and components. I’ve included both live, responsive components as well as their codes under my components section. These components are coded in bootstrap and will be used to create consistency throughout the pages of my portfolio.

I created this design system to help express my personal brand in an interactive form. I will use my site to showcase projects I’ve worked on to potential employers or collaborators. I’d like visitors of the site to immediately gain a sense of my personality and design style when viewing my projects. I believe I am a designer who combines creativity and strategy in modern, clean designs. It’s my hope that this design system reflects these attributes.

Please use the links in the menu to navigate between sections.

Buttons

Buttons are to be used to help guide user actions. When on darker backgrounds, the light versions are to be used. As white and red can sometimes have accessibility issues due to low contrast, the hover states for each button use subtle changes that maintain sufficient contrast.


Use as primary CTA buttons.

Alternate styles to be used on dark backgrounds.

Code

    color:white;	
    background-color:#CD3333;

  }

    background-color:#951A1A;
